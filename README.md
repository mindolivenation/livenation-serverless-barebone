aws access keys<br/>
python 3.7<br/>
node >= v10.15.2<br/>
npm >= 6.4.1<br/>
pipenv, version 2018.11.26<br/>
pip >= 19.0.3<br/>

## How to deploy this project to AWS:

sls deploy --account {replace with your account name} --stage {replace with your stage name}
